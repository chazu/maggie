String

Inherits from: Object

An immutable sequence of characters.

Strings are created with single-quote literals: 'hello'. Concatenate with the comma operator. Compare with =, <, >, <=, >=. Indices are 0-based.

Test 
'hello' size >>> 5
'hello', ' world' >>> 'hello world'
'hello' at: 0 >>> $h
Example 
"Build a greeting"
name := 'Maggie'.
'Hello, ', name, '!'
