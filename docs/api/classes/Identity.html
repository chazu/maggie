Identity

Inherits from: Object
Instance variables: priv, pub

Ed25519-based identity, with a public key used as the stable node name and a private seed used to sign messages. Thin wrapper around the Ed25519 and Sha256 primitives.

Byte arrays in Maggie are represented as Strings (which can hold arbitrary bytes). The priv field is the 32-byte Ed25519 seed; the pub field is the 32-byte public key.

Test 
id := Identity generate.
id publicKey size >>> 32
id privateKey size >>> 32
(Identity verify: (id sign: 'hello') data: 'hello' pub: id publicKey) >>> true
Test 
id := Identity generate.
sig := id sign: 'message'.
(Identity verify: sig data: 'tampered' pub: id publicKey) >>> false
Test 
(Sha256 hash: '') >>> 'e3b0c44298fc1c149afbf4c8996fb92427ae41e4649b934ca495991b7852b855'
(Sha256 hash: 'abc') >>> 'ba7816bf8f01cfea414140de5dae2223b00361a396177a9cb410ff61f20015ad'

bytes := 'deadbeef' fromHex.
bytes size >>> 4
bytes asHex >>> 'deadbeef'
