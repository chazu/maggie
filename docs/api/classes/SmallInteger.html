SmallInteger

Inherits from: Object

Integer values represented as tagged pointers for efficient arithmetic.

SmallInteger is the primary numeric type in Maggie. Arithmetic operators (+, -, *, /, \\, //), comparisons (<, >, <=, >=, =), and iteration (timesRepeat:, to:do:, to:by:do:) are implemented as primitives for performance. The methods below provide higher-level numeric operations built on those primitives.

Test 
3 + 4 >>> 7
10 - 3 >>> 7
6 * 7 >>> 42
10 / 3 >>> 3
10 \\ 3 >>> 1
Example 
"Smalltalk-style iteration"
| sum |
sum := 0.
1 to: 100 do: [:i | sum := sum + i].
sum
