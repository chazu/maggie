Flag

Inherits from: Object
Instance variables: name, type, default, doc, required

A value object describing one command-line flag: its name, type symbol (#string, #bool, or #int), default value, documentation string, and whether the flag is required.

Cli::Flag instances are consumed by Cli::Command>>flag: as an alternative to calling flag:type:default:doc: with four positional arguments. They are immutable after construction — treat them like tuples.

Test 
spec := Cli::Flag name: 'name' type: #string default: 'world' doc: 'Who to greet'.
spec name >>> 'name'
spec type >>> #string
spec default >>> 'world'
spec doc >>> 'Who to greet'
spec required >>> false
Test 
spec := Cli::Flag name: 'token' type: #string default: '' doc: 'API token' required: true.
spec required >>> true

Build a flag spec, explicitly choosing whether the flag is required. Required flags still have a default value — the required bit is advisory and consulted by higher-level validation code.

equals:

Value-object equality. Two Cli::Flag specs are equals:-equal when every ivar — name, type, default, doc, required — matches.

The binary = selector is reserved by the VM's optimized dispatch and always uses identity comparison for user classes, so we expose value equality via a keyword selector that routes through regular method lookup.

Equal when all fields match:

Test 
a := Cli::Flag name: 'foo' type: #string default: 'bar' doc: 'd'.
b := Cli::Flag name: 'foo' type: #string default: 'bar' doc: 'd'.
(a equals: b) >>> true

Reflexive — an instance is equal to itself:

Test 
a := Cli::Flag name: 'foo' type: #string default: 'bar' doc: 'd'.
(a equals: a) >>> true

Symmetric — if a equals: b then b equals: a:

Test 
a := Cli::Flag name: 'foo' type: #string default: 'bar' doc: 'd'.
b := Cli::Flag name: 'foo' type: #string default: 'bar' doc: 'd'.
(a equals: b) = (b equals: a) >>> true

Unequal when names differ:

Test 
a := Cli::Flag name: 'foo' type: #string default: 'bar' doc: 'd'.
b := Cli::Flag name: 'FOO' type: #string default: 'bar' doc: 'd'.
(a equals: b) >>> false

Unequal when types differ:

Test 
a := Cli::Flag name: 'foo' type: #string default: 'bar' doc: 'd'.
b := Cli::Flag name: 'foo' type: #bool default: 'bar' doc: 'd'.
(a equals: b) >>> false

Unequal when defaults differ:

Test 
a := Cli::Flag name: 'foo' type: #string default: 'bar' doc: 'd'.
b := Cli::Flag name: 'foo' type: #string default: 'baz' doc: 'd'.
(a equals: b) >>> false

Unequal when docs differ:

Test 
a := Cli::Flag name: 'foo' type: #string default: 'bar' doc: 'd1'.
b := Cli::Flag name: 'foo' type: #string default: 'bar' doc: 'd2'.
(a equals: b) >>> false

Unequal when required bits differ:

Test 
a := Cli::Flag name: 'foo' type: #string default: 'bar' doc: 'd'.
b := Cli::Flag name: 'foo' type: #string default: 'bar' doc: 'd' required: true.
(a equals: b) >>> false

Comparing to non-Cli::Flag values answers false rather than raising:

Test 
a := Cli::Flag name: 'foo' type: #string default: 'bar' doc: 'd'.
(a equals: 'foo') >>> false
Test 
a := Cli::Flag name: 'foo' type: #string default: 'bar' doc: 'd'.
(a equals: nil) >>> false
Source 
method: equals: other ^<Boolean> [
      other isNil ifTrue: [^false].
      (other class name = self class name) ifFalse: [^false].
      name = other name ifFalse: [^false].
      type = other type ifFalse: [^false].
      default = other default ifFalse: [^false].
      doc = other doc ifFalse: [^false].
      required = other required ifFalse: [^false].
      ^true
  ]
hash

Hash consistent with equals:: derived from the flag's name. Equal flags hash equal; unequal flags may also hash equal (the standard Smalltalk convention — weaker than equality).

Test 
a := Cli::Flag name: 'foo' type: #string default: 'bar' doc: 'd'.
b := Cli::Flag name: 'foo' type: #string default: 'bar' doc: 'd'.
a hash = b hash >>> true
Test 
a := Cli::Flag name: 'foo' type: #string default: 'bar' doc: 'd'.
a hash class name >>> #SmallInteger

printOn:

Emit the same debug representation as printString onto a stream-style collector. The collector only needs to respond to nextPutAll: — anything that behaves like an appending stream works. printOn: is a trivial delegation to printString, so the textual assertions on printString above cover its output exhaustively.

Source 
method: printOn: aStream [
      aStream nextPutAll: self printString
  ]

printString

Return a debug string for the flag in the form Flag name=foo type=#string default=bar. Required flags get an additional required=true suffix.

Test 
s := (Cli::Flag name: 'foo' type: #string default: 'bar' doc: 'd') printString.
(s indexOfSubstring: 'Flag ') >= 0 >>> true
Test 
s := (Cli::Flag name: 'foo' type: #string default: 'bar' doc: 'd') printString.
(s indexOfSubstring: 'name=') >= 0 >>> true
Test 
s := (Cli::Flag name: 'foo' type: #string default: 'bar' doc: 'd') printString.
(s indexOfSubstring: 'type=') >= 0 >>> true
Test 
s := (Cli::Flag name: 'foo' type: #string default: 'bar' doc: 'd') printString.
(s indexOfSubstring: 'default=') >= 0 >>> true
Test 
s := (Cli::Flag name: 'x' type: #bool default: false doc: '' required: true) printString.
(s indexOfSubstring: 'required=true') >= 0 >>> true

Non-required flags should NOT include the required= suffix:

Test 
s := (Cli::Flag name: 'foo' type: #string default: 'bar' doc: 'd') printString.
(s indexOfSubstring: 'required=') >>> -1
