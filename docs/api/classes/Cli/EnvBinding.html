class applyBindingsFor:to:

Apply every binding registered for aCommand to aTarget. Answers the first Failure encountered, or aTarget on success. Called from Cli::Command>>applyEnvBindings.

Source 
method: applyBindingsFor: aCommand to: aTarget [
      | bindings result |
      bindings := self bindingsFor: aCommand.
      bindings do: [:b |
          result := b applyTo: aTarget.
          (result notNil and: [result isFailure]) ifTrue: [^result]
      ].
      ^aTarget
  ]

Source 
method: coerce: aString <String> type: aType <Symbol> [
      | trimmed parsed |
      aString isNil ifTrue: [^nil].
      trimmed := aString trimBoth.

      aType = #string ifTrue: [^trimmed].

      aType = #integer ifTrue: [
          parsed := trimmed asInteger.
          parsed isNil ifTrue: [
              Error signal: 'Cli::EnvBinding: cannot coerce ', trimmed printString, ' to #integer'
          ].
          ^parsed
      ].

      aType = #bool ifTrue: [
          | lower |
          lower := trimmed asLowercase.
          ^(lower = 'true') or: [(lower = '1') or: [lower = 'yes']]
      ].

      aType = #duration ifTrue: [
          "Delegate to Duration parse: when present; otherwise pass the trimmed
          string through unchanged. Keeping this graceful lets EnvBinding land
          before a Duration class is available without breaking callers."
          Duration isNil ifTrue: [^trimmed].
          ^Duration parse: trimmed
      ].

      Error signal: 'Cli::EnvBinding: unknown type ', aType printString
  ]

class register:for:

Associate aBinding with aCommand. Cli::Command>>envBinding: routes here because NaN-boxed command instances cannot hold Maggie ivars of their own.

Source 
method: register: aBinding for: aCommand [
      | list |
      commandBindings isNil ifTrue: [ commandBindings := Dictionary new ].
      list := commandBindings at: aCommand ifAbsent: [
          | l |
          l := ArrayList new.
          commandBindings at: aCommand put: l.
          l
      ].
      list add: aBinding.
      ^aBinding
  ]

method: applyRawValue: aString to: aTarget [
      | coerced |
      (aString isNil or: [aString trimBoth isEmpty]) ifTrue: [
          required ifTrue: [
              ^Failure with: envName, ' is required but not set'
          ].
          default ifNotNil: [
              aTarget perform: setterSelector with: default
          ].
          ^aTarget
      ].
      coerced := self class coerce: aString type: type.
      aTarget perform: setterSelector with: coerced.
      ^aTarget
  ]
applyTo:

Apply the binding to aTarget:

- if the environment variable resolves to an empty string and required is true, answer Failure with: <reason> without touching the target - if the value is empty and the binding has a default, invoke aTarget perform: setterSelector with: default - otherwise coerce the raw value per type and invoke the setter

Separated from the env read so tests can drive the logic directly with applyRawValue:to:.

Default fallback with an unset env var:

Test 
target := ArrayList new.
binding := Cli::EnvBinding envName: 'PP_ENVBINDING_MISSING_XYZ_9f2a' setter: #add: type: #string default: 'fallback'.
binding applyTo: target.
(target at: 0) >>> 'fallback'

Required failure when the env var is unset:

Test 
target := ArrayList new.
binding := Cli::EnvBinding envName: 'PP_ENVBINDING_MISSING_XYZ_9f2a' setter: #add: type: #string default: nil required: true.
result := binding applyTo: target.
result isFailure >>> true
target size >>> 0

Reading a present env var -- HOME is set by the OS on every supported platform. We only assert structural properties of the result so the test is stable across machines:

Test 
target := ArrayList new.
binding := Cli::EnvBinding envName: 'HOME' setter: #add: type: #string.
binding applyTo: target.
target size >>> 1
(target at: 0) class name >>> #String

# ===========================================================================
# Cli::Command extensions
# ---------------------------------------------------------------------------
# `envBinding:` and `applyEnvBindings` live here (rather than in Command.mag)
# because bindings cannot be stored on NaN-boxed Cli::Command instances
# directly -- we keep them on the EnvBinding class-side in a Dictionary
# keyed by the command value. `applyEnvBindings` is intended to be called
# from a custom `execute` wrapper before the run block fires. In practice a
# CLI application writes:
#
#     cmd := Cli::Command new: 'mycmd'.
#     cmd envBinding: (Cli::EnvBinding envName: 'PP_DEMO' setter: #demo: type: #integer).
#     cmd run: [:args | cmd applyEnvBindings to: cfg. ... ].
#
# A future refactor of `cli_primitives.go` may move bindings into the Go
# wrapper and call `applyEnvBindings` inside `execute` automatically. Until
# then, callers are responsible for invoking it.
# ===========================================================================
