class dictText:

Print a Dictionary as aligned key value lines. Iteration order follows Dictionary keys, which Maggie does not guarantee — callers that need a specific field order should use dictText:keys:.

method: dictText: aDict [

class dictText:keys:

Print a Dictionary as aligned key value lines in the order given by keyOrder. Keys in keyOrder that the dict does not contain are skipped; keys in the dict not in keyOrder are not printed.

Source 
method: dictText: aDict keys: keyOrder [
      | maxLen |
      keyOrder isEmpty ifTrue: [^self text: '(empty)'].
      maxLen := 0.
      keyOrder do: [:k |
          (aDict includesKey: k) ifTrue: [
              k size > maxLen ifTrue: [maxLen := k size]
          ]
      ].
      keyOrder do: [:k |
          (aDict includesKey: k) ifTrue: [
              | line val |
              val := aDict at: k.
              line := (self pad: k to: maxLen), '  ', (self formatValue: val).
              self text: line
          ]
      ]
  ]

class result:

Render a result value using the active format. Dictionaries print as aligned key/value lines in text mode; any value prints as JSON in json mode.

Source 
method: result: aValue [
      self isJson
          ifTrue: [self json: aValue]
          ifFalse: [
              (aValue class name = #Dictionary)
                  ifTrue: [self dictText: aValue]
                  ifFalse: [self text: aValue printString]
          ]
  ]
class resultOrError:

Render a Result-like value: Failures print the wrapped error (via error), Successes delegate to result:. Answers true on success, false on failure so the caller can branch on the return.

Source 
method: resultOrError: aValue [
      (aValue isFailure) ifTrue: [
          self isJson
              ifTrue: [self json: (Dictionary new at: 'error' put: aValue error; yourself)]
              ifFalse: [self error: aValue error printString].
          ^false
      ].
      self result: aValue.
      ^true
  ]
class roundTripExamples

Class-side contract: the two supported formats round-trip through format: / format without surprise.

Test 
Cli::Output format: 'json'.
Cli::Output format >>> 'json'
Test 
Cli::Output format: 'json'.
Cli::Output isJson >>> true
Test 
Cli::Output format: 'text'.
Cli::Output isJson >>> false

Padding right-justifies strings up to the requested column width:

Test 
Cli::Output pad: 'hi' to: 5 >>> 'hi   '
Test 
Cli::Output pad: 'hello' to: 3 >>> 'hello'

formatValue: renders nil as the conventional (nil) placeholder, strings verbatim, and non-strings via printString:

Test 
Cli::Output formatValue: nil >>> '(nil)'
Cli::Output formatValue: 'hi' >>> 'hi'
Cli::Output formatValue: 42 >>> '42'

JSON round-trip: a Dictionary encoded via Json encodePretty: and decoded via Json decode: preserves its contents. This is the contract the instance-side printJson: relies on.

Test 
d := Dictionary new at: 'n' put: 1; at: 'name' put: 'ok'; yourself.
encoded := Json encodePretty: d.
decoded := Json decode: encoded.
(decoded at: 'n') >>> 1
Test 
d := Dictionary new at: 'n' put: 1; at: 'name' put: 'ok'; yourself.
decoded := Json decode: (Json encodePretty: d).
(decoded at: 'name') >>> 'ok'

JSON round-trip of an Array of Dictionaries — the shape the table:headers:keys: method prints in JSON mode — preserves keys and values:

Test 
rows := Array with: (Dictionary new at: 'name' put: 'Alice'; yourself).
decoded := Json decode: (Json encodePretty: rows).
decoded size >>> 1
((decoded at: 0) at: 'name') >>> 'Alice'

class table:headers:keys:

Print a table with the given headers. Each row is a Dictionary keyed by keys; headers provides the display labels in the same order as keys. In JSON mode the rows are printed directly.

Source 
method: table: rows headers: headers keys: keys [
      | widths headerLine |
      self isJson ifTrue: [^self json: rows].

      rows isEmpty ifTrue: [^self].

      "Calculate column widths — route through formatValue: so strings
       render without Smalltalk quotes"
      widths := headers collect: [:h | h size].
      rows do: [:row |
          0 to: keys size - 1 do: [:i |
              | val |
              val := self formatValue: (row at: (keys at: i) ifAbsent: ['']).
              (val size > (widths at: i)) ifTrue: [widths at: i put: val size]
          ]
      ].

      "Print header"
      headerLine := ''.
      0 to: headers size - 1 do: [:i |
          headerLine := headerLine, (self pad: (headers at: i) to: (widths at: i)), '  '
      ].
      self text: headerLine trimBoth.

      "Print rows"
      rows do: [:row |
          | line |
          line := ''.
          0 to: keys size - 1 do: [:i |
              line := line, (self pad: (self formatValue: (row at: (keys at: i) ifAbsent: [''])) to: (widths at: i)), '  '
          ].
          self text: line trimBoth
      ]
  ]

printTable:columns:

Print a table.

rows is a sequence of Dictionaries. aColumnSpec is an Array of two-element arrays #(headerString rowKey); the header string is the display label and the row key selects the value from each row.

Example 
| rows cols |
rows := Array with: (Dictionary new at: 'name' put: 'Alice'; yourself).
cols := #(#('NAME' 'name')).
Cli::Output new printTable: rows columns: cols

The column spec decomposes cleanly into parallel header/key arrays:

Test 
cols := #(#('NAME' 'name') #('STATUS' 'status')).
headers := cols collect: [:pair | pair at: 0].
keys := cols collect: [:pair | pair at: 1].
(headers at: 0) >>> 'NAME'
(headers at: 1) >>> 'STATUS'
(keys at: 0) >>> 'name'
(keys at: 1) >>> 'status'

Row data indexed by the column keys survives a Dictionary round-trip, which is the minimal contract the formatter relies on:

Test 
row := Dictionary new at: 'name' put: 'Alice'; at: 'status' put: 'ok'; yourself.
(row at: 'name') >>> 'Alice'
(row at: 'status') >>> 'ok'
Source 
method: printTable: rows columns: aColumnSpec [
      | headers keys |
