Command

Inherits from: Object

A command-line command. Cli::Command is the Smalltalk-idiomatic facade on top of the Go-side cobra bridge (see vm/cli_primitives.go). It provides Unix-style subcommands, flags, help text, and a Maggie block that runs when the command is invoked.

Primitive-provided selectors are declared as <primitive> stubs so their documentation shows up in introspection; the real implementations live on the Go side. Methods without a primitive stub are pure-Maggie helpers that compose those primitives into higher-level selectors such as named:doc:run: and flag:type:default:doc:.

Lifecycle

Example 
"A minimal command: one run block, no flags."
(Cli::Command named: 'greet' doc: 'Say hello' run: [:args |
    'Hello, world!' println.
    0
]) execute

Flags

Example 
"Declare flags, then read them from inside the run block."
cmd := Cli::Command named: 'greet' doc: 'Say hello'.
cmd flag: 'name' type: #string default: 'world' doc: 'Who to greet'.
cmd run: [:args |
    ('Hello, ', (cmd flagValue: 'name'), '!') println.
    0
].
cmd execute

Subcommands

Example 
"Compose a parent command with children."
root := Cli::Command named: 'mytool' doc: 'My multi-command tool'.
build := Cli::Command named: 'build' doc: 'Build the project' run: [:args | 0].
test := Cli::Command named: 'test' doc: 'Run tests' run: [:args | 0].
root registerSubcommand: build.
root registerSubcommand: test.
root execute

Exit-code convention: a SmallInteger return from the run block becomes the process exit code. An unhandled exception yields exit code 1. Any other return value yields exit code 0. See the Go primitive for details.

applyEnvBindings:

Apply every registered binding to aTarget. Called by CLI applications from inside their run block (or from a subclass override of execute) before the user's logic fires. Answers the first Failure encountered or aTarget on success.

cmd applyEnvBindings: target.
(target at: 0) >>> 'fallback'

envBinding:

Attach aBinding to the receiver. The binding is stored on the Cli::EnvBinding class-side registry keyed by the command's identity. Answers the command so calls chain.

flag:

Register a flag from a Cli::Flag value object. Unpacks its name/type/default/doc and delegates to flag:type:default:doc:. The required bit on the spec is advisory and not enforced by the primitive; higher-level validation code should consult it directly.

Test 
spec := Cli::Flag name: 'verbose' type: #bool default: false doc: 'Chatty'.
cmd := Cli::Command named: 'demo'.
cmd flag: spec.
cmd flagTypeOf: 'verbose' >>> #bool
Source 
method: flag: aFlagSpec <Cli::Flag> ^<Cli::Command> [
      ^self
          flag: aFlagSpec name
          type: aFlagSpec type
          default: aFlagSpec default

flag:type:default:doc:

Register a flag by name, type (#string, #bool, or #int), default value, and doc string. Dispatches to the correct primitive add*Flag:default:doc: based on the type symbol.

method: flag: aName <String> type: aType <Symbol> default: aDefault doc: aDoc <String> ^<Cli::Command> [
      aType = #string ifTrue: [^self addStringFlag: aName default: aDefault doc: aDoc].
      aType = #bool ifTrue: [^self addBoolFlag: aName default: aDefault doc: aDoc].
      (aType = #int or: [aType = #integer]) ifTrue: [^self addIntFlag: aName default: aDefault doc: aDoc].
      ^self error: 'Cli::Command flag: unknown flag type ', aType printString
  ]
flagValue:

Return the parsed value of a declared flag, dispatching to stringFlag:, boolFlag:, or intFlag: based on how the flag was registered. Returns nil if the flag was never declared.

method: flagValue: aName <String> [
      | kind |
      kind := self flagTypeOf: aName.
      kind isNil ifTrue: [^nil].
      kind = #string ifTrue: [^self stringFlag: aName].
      kind = #bool ifTrue: [^self boolFlag: aName].
      kind = #int ifTrue: [^self intFlag: aName].
      ^self error: 'Cli::Command flagValue: unknown flag type ', kind printString
  ]
